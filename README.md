# Kinship Inference

This repository contains MATLAB implementation of experiments done in the article: 
**The effect of kinship in re-identification attacks against genomic data sharing beacons** 
[Link to paper](https://academic.oup.com/bioinformatics/article/36/Supplement_2/i903/6055924)

## How to Run
To run the experiments, execute Inference.m code. Note that datasets are not shared.
